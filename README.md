A simple and interactive web application that displays 20 dog breeds with fun facts, featuring a search function, dark mode toggle, and a back button to reset the search view.

This project is built using HTML, CSS, and JavaScript only, making it beginner-friendly and easy to deploy on GitHub Pages.

## Features

  - Search Dog Breeds
  - Search by breed name (case-insensitive)
  - Displays matching dog cards instantly
  - Shows ‚ÄúNo results found‚Äù if no match exists

  - Back Button (Overlay)
  - Appears only after searching
  - Clears the search input
  - Restores the original 20-dog grid layout

  - Dark Mode Toggle
  - Switch between light and dark themes
  - Smooth transition for better user experience

  - Fixed Dog Images
  - Each breed has a fixed image
  - Images do not change when searching or navigating back
